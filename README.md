# CRYPDASH [Cryptocurrency Portfolio Tracking Application]

**CRYPDASH** is a cryptocurrency portfolio tracking web application that enables users to monitor crypto assets, analyze performance, and visualize portfolio distribution using real-time market data.

The project focuses on **data visualization, API integration, and intuitive UI design** to support informed, data-driven decisions.
